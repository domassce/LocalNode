# LocalNode

LocalNode is a local network file hosting server built with C# and Avalonia UI. It allows you to transform any computer into a file-sharing node within a local area network without relying on external cloud services.



## Features

* Node Dashboard: Real-time server status monitoring, active hosting statistics, and storage metrics.
* Integrated Explorer: Manage root directories.
* Path Intelligence: UI implementation featuring smart path truncation and informative tooltips for deep directory structures.
* Network Client: Connect to other LocalNode instances across the same Wi-Fi or Ethernet network.
